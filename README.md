# synDcat - Farm Management System

A web-based platform to connect farmers and agro-product buyers. Built with **Jakarta EE (Servlets + JSP)**, **MySQL**, and **Bootstrap**.

## Features
- User Signup & Login (BCrypt password hashing)
- Farmer Registration & Profile Management
- Add / View / Update / Delete Agro Products
- Responsive UI with Bootstrap 5 & AOS animations
- Database triggers for logging farmer changes

## Tech Stack
- **Backend:** Jakarta Servlets, JSP, JDBC
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript, Font Awesome, AOS
- **Database:** MySQL (with triggers)
- **Security:** BCrypt for password hashing
- **Server:** Apache Tomcat 10+
